## 💼 Project-Based Experience

### 🚀 DevConnect — Social Platform for Developers

**Role:** Co-creator / Full Stack Developer  
**Stack:** Angular · Laravel · MySQL · Sanctum · Cypress · PHPUnit · Docker

DevConnect is a full stack web application designed as a social network for developers, including authentication, posts, comments, likes, user profiles, follows, saved posts and user search.

Key contributions:

- Built features across frontend and backend
- Implemented SPA architecture with Angular consuming a Laravel REST API
- Worked with Laravel Sanctum for session-based authentication
- Developed and stabilized E2E testing with Cypress
- Added backend coverage with PHPUnit
- Collaborated through branches, pull requests and code reviews
- Improved maintainability, testing strategy and project documentation

### 🧪 Testing Coverage

- 74 E2E tests with Cypress  
- 75 backend tests with PHPUnit  
- 30 frontend tests with Karma/Jasmine
